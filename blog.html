Question-1: What are the different ways to select an element in the DOM?

You can select an element with following DOM methods.
1.getElementById();
2.getElementsByClassName();
3. getElementsByTagName();
4. querySelector();
5. querySelectorAll();

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML will give all HTML elements of the selected element;
innerText will give only text content of the selected element;
textContent will give HTML element and text nodes of the selected element including comments.

Question-3: What is event delegation in the DOM?

Event delegation is use for propagation and that allows us to handle events for its child elements. Even the children are added latter.

Question-4: What is event bubbling in the DOM?

Event bubbling is use for event propagation where event start from its target and bubbles up to its ancestor elements.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement() using this method you can create element.
appendChild() using this method you can add element in DOM.
remove(), removeChild() using these methods you can remove element from DOM.
